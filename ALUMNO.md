## ¿Qué es Git?

Git es un sistema de control de versiones que permite guardar los cambios que
hacemos en un proyecto.

## Por qué necesitamos Git

Porque cuando varias personas trabajan en un mismo proyecto se pueden perder
archivos o sobrescribir cambios importantes.

## Los 3 conceptos clave de Git

1. **Repositorio:** Es la carpeta del proyecto donde Git guarda todos los archivos
   y el historial.

2. **Commit:** Es un guardado de los cambios realizados en un momento específico.

3. **Rama:** Es una línea de trabajo independiente que permite hacer cambios sin
   afectar la versión principal del proyecto.

## Los comandos que aprendí

- `git init` — Inicializar un repositorio
- `git add` — Preparar archivos para guardar
- `git commit` — Guardar cambios con un mensaje
- `git push` — Subir cambios a GitHub
- `git pull` — Descargar cambios desde GitHub
- `git status` — Ver el estado del repositorio
